# Tips & Tricks

Getting the most out of the playground

---

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `⌘/` | Toggle slides |
| `⌘S` | Save code |
| `⌘Enter` | Run code |
| `Esc` | Close panels |

---

## Sharing Demos

1. Make changes to code
2. Click the QR button
3. Share link or scan code

Your code is saved in the URL!

---

## Browser Support

Check the compatibility badge before trying an API.

- ✅ **Green** = Fully supported
- ⚠️ **Yellow** = Partial support
- ❌ **Red** = Not supported

Some APIs require HTTPS or specific flags.
